# 暗林信号

当前项目严格按 PDF 的 M0-M8 顺序开发。

## 当前阶段

M3：信号系统。

已完成：

- M0 项目骨架。
- M1 网格地图与移动。
- M2 暗能、回合与基础 NPC。
- M3 信号系统、NPC 默认隐藏、移动信号、扫描信号、攻击信号、信号衰减。

## 运行方式

1. 使用 Godot 4.6 打开项目目录：`D:\GameC\dark\dark-f`
2. 运行主场景：`res://scenes/Main.tscn`

## M3 验收方式

1. 运行后 NPC 默认不可见。
2. 玩家移动后生成蓝色移动信号。
3. 点击“扫描”后点击地图，消耗暗能并生成紫色扫描信号。
4. 点击“攻击”后点击预测格，消耗暗能并生成红色攻击信号。
5. 点击“结束回合”后，NPC 逐个行动；NPC 移动会留下信号。
6. 信号会随新回合衰减并消失。
7. 如果攻击命中隐藏 NPC，NPC 数量减少并生成死亡信号。
