# M4 地图机制开发日志

日期：2026-05-02

## 本阶段目标

严格按照 PDF 中 M4 的要求，在 M3 信号系统基础上加入地图驱动机制：资源点、技能点、静默区、回声区和缩圈黑域。同时补充基础攻击反馈，让攻击不再只是日志变化。

## 已完成内容

1. 瓦片图集扩展：
   - 保留普通深空格。
   - 保留障碍格。
   - 新增静默区瓦片。
   - 新增回声区瓦片。
   - 新增黑域瓦片。
   - 这些区域格子使用 TileSet 坐标，后续可继续往手铺地图方向扩展。

2. 资源点：
   - 从 `game_config.json` 读取资源点配置。
   - 地图上显示黄色菱形点位。
   - 玩家站在资源点上可点击“采集”。
   - 采集后获得暗能，资源点消失，并产生黄色采集信号。

3. 技能点：
   - 从 `game_config.json` 读取技能点配置。
   - 地图上显示紫色菱形点位，并显示 Lv.1-Lv.4。
   - 玩家站在技能点上可点击“拾取”。
   - 拾取消耗暗能，技能点消失，并产生紫色技能波动。
   - 当前阶段不发放技能，三选一技能卡放到 M5。

4. 技能配置预留：
   - 新增 `configs/skill_config.json`。
   - 预留技能池、技能 id、中文名称、类型、消耗、冷却、范围、信号强度、信号持续时间和描述。
   - M5 会基于该配置继续实现技能三选一、技能槽、释放和冷却。

5. 静默区：
   - 静默区由蓝色瓦片块组成。
   - 玩家或 NPC 在静默区移动时不生成移动轨迹。
   - HUD 提示和日志会说明“静默区未生成移动信号”。

6. 回声区：
   - 回声区由橙色瓦片块组成。
   - 单位在回声区移动或行动时会生成公开信号。
   - 公开信号不受玩家扫描范围限制，全图可见。

7. 缩圈黑域：
   - 新增缩圈倒计时。
   - 到达缩圈时，地图外圈按格子变为黑域。
   - 玩家位于黑域内时先扣暗能，暗能不足时扣 HP。
   - 黑域腐蚀强度按阶段递增，当前规则为基础腐蚀 + 每阶段递增。
   - NPC 位于黑域内时同样会被腐蚀，HP 归零后被消灭并生成死亡信号。
   - HUD 顶部显示缩圈倒计时，右下角小地图面板显示安全区文字信息。

8. 攻击反馈：
   - 攻击后目标格出现爆闪反馈。
   - 命中时偏红，未命中时偏橙。
   - 攻击源附近出现短暂扩散波纹。

9. 规则测试：
   - 新增 `tests/m4_rules_test.gd`。
   - 覆盖静默区不留移动信号、回声区公开信号、资源点采集、技能点拾取、缩圈黑域腐蚀。

## 当前验收标准

1. 运行主场景后，地图可见资源点、技能点、静默区和回声区。
2. 玩家移动到资源点并点击“采集”后，暗能增加，资源点消失，出现黄色采集信号。
3. 玩家移动到技能点并点击“拾取”后，暗能减少，技能点消失，出现紫色技能波动。
4. 玩家或 NPC 在静默区移动时不留下移动轨迹。
5. 玩家或 NPC 在回声区行动时生成公开广播信号。
6. 缩圈推进后，外圈格子变为黑域。
7. 玩家在黑域内结算时先扣暗能，暗能不足时扣 HP，且黑域阶段越高腐蚀越强。
8. NPC 在黑域内结算时也会被腐蚀，死亡时日志显示“被黑域吞没，已消灭”。
9. 所有 NPC 被清除后会进入胜利结算，中央显示“胜利”，并禁用行动按钮。
10. 攻击无论命中或未命中都有可见反馈。
