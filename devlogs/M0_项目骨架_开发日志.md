# M0 项目骨架开发日志

日期：2026-05-02

## 本阶段目标

严格按照 PDF 中的 M0 要求创建《暗林信号》的 Godot 4 项目骨架。本阶段只做项目能运行、能显示基础网格和基础 HUD，不提前实现移动、NPC、信号、资源、技能、缩圈等后续里程碑内容。

## 已完成内容

1. 建立基础目录：
   - `assets/`
   - `configs/`
   - `scenes/`
   - `scripts/`
   - `ui/`
   - `devlogs/`

2. 创建主场景：
   - `res://scenes/Main.tscn`

3. 创建核心模块：
   - `GameController`：主流程入口，负责连接配置、回合状态、地图和 HUD。
   - `ConfigLoader`：读取 `configs/game_config.json`。
   - `GridMap`：场景节点名称为 `GridMap`，实际脚本类名为 `DarkSignalGridMap`，用于避免和 Godot 内置 3D `GridMap` 类型冲突。
   - `TurnManager`：保存当前回合、暗能和暗能上限。
   - `HUD`：显示标题、回合和暗能。

4. 创建 JSON 配置：
   - `res://configs/game_config.json`
   - 当前包含地图宽高、瓦片尺寸、初始回合、初始暗能和暗能上限。

5. 创建瓦片化网格：
   - 使用 `TileMapLayer` 实现 24 x 16 网格。
   - 使用 `res://assets/m0_grid_atlas.svg` 作为 M0 占位瓦片。
   - 使用 `res://scenes/m0_grid_tileset.tres` 作为 TileSet。

## 当前验收标准

打开项目并运行后，应看到：

1. 主场景可以启动。
2. 画面中显示 24 x 16 的基础网格。
3. 顶部 HUD 显示游戏标题。
4. 顶部 HUD 显示当前回合。
5. 顶部 HUD 显示暗能数值。
